BB360MSX0edition.DSK
　このファイルはMSXのディスクイメージになってます。
　ダウンロードしたらMSX0用SDカードのDSKフォルダに入れて、Setup Utilityからこのディスクイメージ(BB360MSX0edition)を選択して下さい。

BBMSX0E.BAS
　
　LOAD"BBMSX0E.BAS"と打ってロードして、RUNしてください。
　黒色のポートにGPIO接続したアナログつまみで向きを定め、青い鳥にタッチでいいねを発射します。
 
BLUEBIRD.BAS
　LOAD"BLUEBIRD.BAS"と打ってロードしてRUNしてください。
　キーボードの左右で向きを決め、スペースキーでいいねを発射します。
　
　
autoBB360MSX0edition.DSK
　このファイルはMSXのディスクイメージになっています。
　MSX0用SDカードのDSKフォルダに入れて、Setup Utilityからこのディスクイメージ(autoBB360MSX0edition)を選択して下さい。
　自動で起動します。
　黒色のポートにGPIO接続したアナログつまみで向きを定め、青い鳥にタッチでいいねを発射します。

SOUNDTESTV1.DSK
　これはDSKイメージです。自動でSOUNDTESTが立ち上がります。

SOUNDTEST0.DSK
　これはDSKイメージです。SOUND0.BASというファイルがSOUNDTEST MSX0 editionになります。
　SOUNDTEST MSX0 editionとの違いは、コードモードの音程の上下をカーソル左右で行うか、アナログつまみで行うかです。
